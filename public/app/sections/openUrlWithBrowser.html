Opening URLs with the system default Browser

Option 1: System.openUrlWithBrowser()

The OpenFin API method System.openUrlWithBrowser() will open URLs with the local systems default browser.

Example:

An example function named openUrlWithBrowser() is provided in this project in .\public\examamples\examples.js. The button below calls the function and will Open our openUrlWithBrowser tutorial in this system default browser.

Open With Browser:

{
 var url = 'http://cdn.openfin.co/docs/javascript/stable/tutorial-System.openUrlWithBrowser.html';  fin.System.openUrlWithBrowser(url)
 .then(() => console.log('Opened URL'))
 .catch(err => console.log(err));
}

References:

API Documentation: System.openUrlWithBrowser()

Option 2: System.launchExternalProcess()

The OpenFin API method System.launchExternalProcess() can be used with just the `path` parameter to launch a URL initialising the HTTP/HTTPS protocol handers, resulting in the systems default browser being initiated.

{
 fin.System.launchExternalProcess({
  path: 'http://www.openfin.co'
 })
 .then(processIdentity => console.log(processIdentity))
 .catch(err => console.log(err));
}
