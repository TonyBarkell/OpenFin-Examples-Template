Opening URLs with the system default browser using System.launchExternalProcess()

Overview:

The OpenFin API method System.launchExternalProcess() can be used with just the `path` parameter to launch a URL initialising the HTTP/HTTPS protocol handers, resulting in the systems default browser being initiated.

Example:

An example function named launchExternalProcessExample() is provided in this project in .\public\examamples\examples.js. The button below calls the function and will Open the 'launchExternalProcess' tutorial in this systems default browser.

Open In Browser

{
  url = 'http://cdn.openfin.co/docs/javascript/stable/System.html#launchExternalProcess'  fin.System.launchExternalProcess({
  path: url
 })
 .then(processIdentity => console.log(processIdentity))
 .catch(err => console.log(err));
}
