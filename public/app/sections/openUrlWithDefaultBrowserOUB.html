Opening URLs with the system default browser using System.openUrlWithBrowser()

Overview:

The OpenFin API method System.openUrlWithBrowser() opens URLs using the local systems default browser.

Example:

An example function named openUrlWithBrowser() is provided in this project in .\public\examamples\examples.js. The button below calls the function and will Open the 'openUrlWithBrowser' tutorial in this systems default browser.

Open In Browser

{
 var url = 'http://cdn.openfin.co/docs/javascript/stable/tutorial-System.openUrlWithBrowser.html';  fin.System.openUrlWithBrowser(url)
 .then(() => console.log('Opened URL'))
 .catch(err => console.log(err));
}
